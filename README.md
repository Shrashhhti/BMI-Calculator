# BMI-Calculator
A BMI Calculator is a simple tool that helps to check whether a person’s weight is healthy or not using height and weight. It shows if someone is underweight, normal, overweight, or obese. It is easy to use and helpful for basic health awareness.
A BMI (Body Mass Index) Calculator is a simple web project that helps users check whether their weight is healthy based on their height and weight. It calculates BMI using a standard formula and shows the health category such as underweight, normal, overweight, or obese.

This project is beginner-friendly and useful for understanding basic input handling, calculations, and logic in web development.

Features

Takes height and weight as input

Calculates BMI instantly

Shows BMI value and health category

Simple and easy-to-use interface

Beginner-friendly project

Technologies Used

HTML

CSS

JavaScript

How BMI is Calculated

BMI = weight (kg) / (height × height in meters)

BMI Categories

Underweight: BMI < 18.5

Normal weight: 18.5 – 24.9

Overweight: 25 – 29.9

Obese: 30 or above

Project Purpose

This project is created for learning basic frontend concepts and practicing JavaScript logic. It is suitable for beginners and can be extended with better UI or additional health features.
